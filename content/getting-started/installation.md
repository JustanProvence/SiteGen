# Installation

## Requirements

- Python 3.12+
- Poetry 2.x
- Podman (for container deployment)

## Install with Poetry

```bash
git clone https://github.com/your-org/sitegen.git
cd sitegen
poetry install
```

## Run the dev server

```bash
poetry run python -m sitegen.main
```

Open `http://localhost:8080` in your browser.
